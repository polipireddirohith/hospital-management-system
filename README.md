## ğŸ“ Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-red: #DC143C;
    --accent-red: #E91E63;
    /* Modify other colors as needed */
}
```

### Adding New Services
1. Copy a service card in `index.html`
2. Update the image, title, and description
3. Add corresponding icon styles in `styles.css`

### Modifying Doctors
1. Update doctor information in `index.html`
2. Replace image URLs with your doctor photos
3. Adjust statistics and specialties
